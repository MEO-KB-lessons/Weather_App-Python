# 🌦️ Дневник погоды (Weather Journal)

Консольное приложение на Python для просмотра и сохранения истории погоды.  
Позволяет получать данные о погоде за прошедшие даты, сохранять их в локальный JSON-файл и просматривать историю для выбранного города.

---

## 📋 Возможности

- Поиск города по названию (геокодинг Open‑Meteo)
- Автоматическое определение временной зоны для города
- Получение погоды на **сегодня** или **любую прошедшую дату**
- Сохранение данных в **JSON-дневник** (не требует базы данных)
- Просмотр истории погоды для текущего города
- Смена города с автоматическим обновлением временной зоны
- Перевод единиц измерения:
  - Скорость ветра: км/ч → м/с
  - Давление: гПа → мм рт. ст.
- Красивое консольное отображение с эмодзи

## 🚀 Установка и запуск

### Клонирование репозитория

```bash
git clone https://github.com/your-username/weather-journal.git
cd weather-journal
```

## Создание виртуального окружения (рекомендуется)

#### Windows
```
python -m venv venv
venv\Scripts\activate
```

#### Linux/Mac
```bash
python3 -m venv venv
source venv/bin/activate
```

### Установка зависимостей
```
pip install -r requirements.txt
```

### Запуск

#### Windows
```
python Weather_App.py
```

#### Linux/Mac
```bash
python3 Weather_App.py
```
